# Criação de uma api para perguntas de kpop.

## instruções para api

Para os recursos da api deve ser usado o padrão controller, service e repository
Exemplo: UserController, UserService, UserRepository

### Controller

Onde fica as os endpoints que serão disponibilizados pela api

### Service

Onde serão executadas as regras de negócio.

### Repository

Onde ficarão os acesos aos ados. Ponde ser arquivos, banco de dados e etc.
